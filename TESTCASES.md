# Тест-кейсы для таск-трекера

## Создание задачи

### Создание задачи через header 

**ID**: 001

**Описание**: Проверка поведения системы при попытке создать новую задачу в таск-трекере через header (статус по умолчанию - backlog).

**Предусловие**: Открыта главная страница https://avito-tech-internship-psi.vercel.app/

**Шаги**:
1. Нажать на кнопку "Создать задачу".
   - ОР: Появляется окно создания задачи

2. Ввести название задачи

3. Ввести описание задачи

4. Нажать на строку "Проект"
   - Появляется выпадающий список проектов

5. Выбрать проект, например, "Редизайн карточки товара"

6. Нажать на строку "Приоритет"
   - Появляется выпадающий список приоритетов

7. Выбрать приоритет, например, "High"

8. Нажать на строку "Исполнитель"
   - Появляется выпадающий список исполнителей

9. Выбрать исполнителя, например, "Екатерина Смирнова"
   - Кнопка "Создать" становится активной

10. Нажать на кнопку "Создать"

**Ожидаемый результат**: Новая задача отображается в списке с указанным заголовком.
